# RepoDoc Mobile

AI-Powered Code Documentation & Repository Intelligence Mobile App built with Flutter.

## Features

- **Code Documenter** - Auto-generate comprehensive code documentation
- **README Generator** - Professional README.md creation
- **API Docs Generator** - OpenAPI-style documentation from code
- **Architecture Analyzer** - Design pattern detection and architecture diagrams
- **Changelog Generator** - Keep a Changelog format auto-generation
- **Material 3 Dark Theme** - Emerald accent with terminal/code monospace theme
- **9 Premium Screens** - Splash, onboarding, home, code documenter, README gen, API docs, architecture, changelog, settings

## AI Backend

- **Endpoint**: `sai.sharedllm.com/v1/chat/completions`
- **Model**: `gpt-oss:120b`
- **No API key required**

## Getting Started

```bash
flutter pub get
flutter run
```

## Tech Stack

- Flutter 3.x with Material 3
- Dark theme with emerald accent + monospace fonts
- HTTP for AI API calls
- Provider for state management
